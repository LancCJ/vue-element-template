# vue-element-template
vue基于element-ui的脚手架

### 技术结构
* vue javascript框架
* vue-router 路由管理
* vuex 状态管理
* axios 基于Promise的HTTP请求客户端
* webpack 打包工具
* Elememt-UI vue组建库

### 后台骨架
* 整体布局(顶部、侧栏、主体)
* 菜单、面包屑
* 路由(路由规则、路由导航钩子)
* 状态/缓存 (登陆状态,用户信息/权限信息)
* 接口请求(拦截器,get post请求)
* 组件(表单,列表,分页等)
* 权限(菜单权限、操作权限、视图权限)
* 日志(错误日志、操作日志、前端埋点)
* 公共页(主页、登陆、修改密码、404、401)
